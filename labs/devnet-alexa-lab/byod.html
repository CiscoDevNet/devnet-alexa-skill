The DevNet Alexa Data Center Skill lab does not require an Amazon Echo. All Alexa Skill development and testing can be done using the developer.amazon.com site.

The Lambda function and S3 AWS services covered in the lab require an AWS account. You can use an AWS free tier account for this lab. How to get an AWS account, free tier or otherwise, is documented on the aws.amazon.com site.
